# IA-Tracker — Reporte de inversión inmobiliaria

Reporte estático autogenerado con datos reales de portales inmobiliarios
(Fincaraiz + Ciencuadras) para **Manizales y Villamaría, Caldas**.

**Ver el reporte:** https://jsebgiraldo.github.io/ia-tracker-reporte/

Incluye subvaloración por barrio, cap rate real, estrato/afinidad y modo por
perfil (inversionista / comprador / arrendatario). Generado por el proyecto
privado IA-Tracker. No es asesoría financiera.
